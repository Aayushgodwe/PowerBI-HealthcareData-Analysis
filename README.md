Data Visualization Project using Power BI
# 🏥 Healthcare Disease Prediction Dashboard

## 🔹 Project Overview

This project focuses on analyzing a healthcare dataset to predict diseases based on various patient attributes. Using data visualization techniques in Power BI, the dashboard provides meaningful insights into patient health trends, disease distribution, and risk factors.

---

## 🔹 Dataset Information

* Dataset Name: Healthcare Disease Prediction Dataset
* Format: CSV
* Description: Contains patient data including age, gender, symptoms, and disease outcomes used for analysis and prediction.

---

## 🔹 Objectives

* Analyze patient health data
* Identify patterns related to diseases
* Visualize trends for better understanding
* Provide insights for early disease prediction

---

## 🔹 Tools & Technologies Used

* Microsoft Power BI
* Microsoft Excel / CSV
* GitHub

---

## 🔹 Dashboard Features

* 📊 Disease distribution analysis
* 📈 Age vs disease trends
* 🧑 Gender-based comparison
* 📌 Key Performance Indicators (KPIs)
* 📉 Interactive charts and filters

---

## 🔹 Key Insights

* Certain age groups are more prone to specific diseases
* Gender plays a role in disease occurrence
* Some symptoms strongly indicate particular diseases
* Early analysis can help in preventive healthcare
